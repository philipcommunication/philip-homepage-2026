# 필립(Philip) 홈페이지 구축 이행 계획서

본 계획서는 '필립' 브랜드의 프리미엄 정체성을 확보하고, 데이터 기반 마케팅 전문성을 강조하는 웹사이트 구축을 목표로 합니다.

## 1. 단계별 상세 계획

### Phase 1: 시각적 완성도 강화 (Visual Excellence)
- **로고 및 디자인 에셋 생성**: `generate_image`를 활용하여 필립의 골드/네이비 톤에 맞는 고품질 로고 및 메인 이미지를 생성합니다.
- **유리질 효과(Glassmorphism) 세밀화**: 각 섹션의 카드 및 헤더에 더 깊이감 있는 블러 효과와 테두리 조명 효과를 적용합니다.
- **타이포그래피 최적화**: 'Pretendard' 또는 'Outfit' 폰트를 사용하여 모던하고 신뢰감 있는 텍스트 환경을 조성합니다.

### Phase 2: 콘텐츠 보강 및 디테일 개선
- **회사소개 고도화**: 단순 플레이스홀더를 제거하고, 실제 브랜드 스토리를 담은 문구로 교체합니다.
- **업무영역 상세화**: 각 서비스별 특장점을 시각적으로 인지하기 쉽게 아이콘 및 짧은 애니메이션을 추가합니다.
- **레퍼런스 섹션**: 마우스 호버 시 상세 정보를 보여주는 인터랙티브 효과를 적용합니다.

### Phase 3: 콘텐츠 고도화 및 데이터 관리 효율화
- **코드 기반 레퍼런스 관리**: `src/data/references.ts`와 같은 데이터 파일을 생성하여, 이미지 경로와 협업 링크를 코드 수정만으로 간편하게 관리할 수 있도록 구조화합니다.
- **연락처 다이렉트 연결**: 문의 폼 대신 전화번호(`tel:`) 및 카카오톡/인스타그램 링크를 직접 활성화하여 실시간 상담 연결을 강화합니다.
- **블로그 메뉴 관리**: 현재는 메뉴에서 숨김 처리하며, 추후 필요 시 `blog.philip.kr` 등 외부 주소 연동을 준비합니다.

### Phase 4: SEO 및 퍼포먼스 최적화
- **메타 데이터 설정**: 각 페이지별 고유 Title 및 Meta Description을 설정하여 검색 엔진 노출을 최적화합니다.
- **이미지 최적화**: Next.js `Image` 컴포넌트와 `webp` 형식을 적극 활용하여 로딩 속도를 극대화합니다.
- **반응형 웹**: 모바일 기기에서의 터치 인터랙션과 레이아웃을 최적화합니다.

## 2. 디자인 테마 가이드
- **핵심 컬러**: 딥 네이비 (#1D3557), 프리미엄 골드 (#B87333), 클린 화이트 (#FFFFFF).
- **분위기**: 신뢰감(Trust), 데이터 기반(Data-driven), 혁신적(Innovative).
- **인터랙션**: 부드러운 스크롤, 미묘한 페이드 인/아웃, 강조 포인트의 글로우 효과.
