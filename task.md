- **프레임워크**: Next.js 14+ (App Router)
- **스타일링**: CSS Modules (Vanilla CSS)
- **상태 관리**: React Hooks
- **데이터 관리**: 정적 JSON/Array 기반 코드 관리 (Supabase 미사용)
