# UX/UI Designer Skill: Design Execution

## 절차
1. **리서치 및 벤치마킹**: 최신 디자인 트렌드(SaaS, Marketing Agency)를 분석하여 필립에 맞는 스타일을 도출합니다.
2. **디자인 시스템 정의**: CSS 변수(Variables)로 사용될 컬러 팔레트와 폰트 크기 세트를 정의합니다.
3. **컴포넌트 설계**: 메인 히어로 섹션, 업무영역 카드, 레퍼런스 슬라이더 등 각 요소의 상세 디자인 가이드를 작성합니다.
4. **인터랙션 명세**: `Framer Motion` 또는 `Vanilla CSS Animation`으로 구현할 효과를 텍스트로 상세히 기술합니다.

## 규칙
- 모든 디자인 제안은 "필립"의 브랜드 가치인 '데이터 기반'을 시각적으로 뒷받침해야 합니다.
- 모바일 우선(Mobile-First) 반응형 레이아웃을 반드시 고려합니다.
- 접근성(Accessibility)을 준수하여 텍스트 대비와 클릭 영역을 확보합니다.
